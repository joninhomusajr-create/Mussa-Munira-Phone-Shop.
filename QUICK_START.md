## 💡 Quick Tips

### To Test Locally
```bash
# Python 3
python -m http.server 8000

# Open browser
http://localhost:8000
```

### To Make Changes
1. Edit HTML file
2. Edit CSS kwa `css/styles.css`
3. Edit JavaScript kwa `js/script.js`
4. Refresh browser (F5)
5. View changes kamaata

### To Add Products
1. Edit `productsData` kwa `js/script.js`
2. Add new object yenye format:
```javascript
{
    id: 7,
    name: "Simu Jina",
    brand: "Brand",
    oldPrice: 2000000,
    newPrice: 1800000,
    ram: "8GB",
    storage: "256GB",
    stock: 5
}
```

### To Change Colors
1. Open `css/styles.css`
2. Find `:root` section mwanzoni
3. Update color codes
4. All colors automatically change!
